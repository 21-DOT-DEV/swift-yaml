## 1. Goal & success criteria

Let a caller change one value in a hand-maintained YAML document and get back text whose **only** difference is that value — a one-line diff, not a reformatted file.

**Roadmap success metrics (L23–29):**
- Changing a value preserves every comment, blank line, and key order; the diff covers only the edited span.
- The result re-parses to the intended new value (oracle: re-parse through yaml-cpp, an independent path).
- A missing/unaddressable path returns a typed error, never a silent no-op.

**Concrete criteria:**
- `try YAMLEditor.set(yaml, at: ["server", "port"], to: 9090)` changes only the `8080`→`9090` bytes; comments/order/other quoting byte-identical; re-parses to `9090`.
- A quoted target (`tls: "off"` → `to: "on"`) stays a quoted string (and must — `on`/`off` are YAML booleans).
- A numeric-looking string (`to: "2"`) is emitted quoted so it re-parses as a string.
- A wrong/unaddressable path, an uneditable target shape, or an anchor/alias document throws a typed `YAMLEditError` naming the path — never a silent no-op.

## 2. Scope

**In scope:** `YAMLEditor.set` (static); `YAMLPath.Component`; the public `YAMLEditor.Value`; navigation + span + splice for **plain and quoted single-line scalar** targets; the `YAMLEditError` taxonomy; tests; one edit per call, first document.

**Out of scope (explicit owners / typed errors):** multi-line/block/folded scalar targets and **null (empty) targets** → `unsupportedValueShape` (a later iteration; writing an absent value is an insertion, not a splice); non-scalar (map/sequence) targets → `notASingleValue` (this covers a flow collection addressed *as a whole* — but a scalar *inside* a flow collection **is** in scope and editable); **documents using anchors/aliases** → `documentUsesAnchorsOrAliases` (D6 — the node tree can't locate an alias safely; a DOM-based, anchor-preserving editor is future); multi-document → not rejected, first document only (A6); `unset` → **feature 2** (reuses this locate-by-path machinery); a dotted-string path convenience and a raw-string replacement → later; preserving the exact original quote glyph (single vs double) on a quoted edit → deferred nice-to-have (result stays correct + minimal; only the glyph may change).

## 3. Design

**Public surface** (names settle in review):
```swift
public enum YAMLEditor {   // caseless namespace (no instances), so it needs no Sendable
    public static func set(_ yaml: String, at path: [YAMLPath.Component], to value: Value) throws -> String

    public enum Value: Sendable, ExpressibleByStringLiteral, ExpressibleByIntegerLiteral,
                       ExpressibleByFloatLiteral, ExpressibleByBooleanLiteral, ExpressibleByNilLiteral {
        // Mirrors YAMLValue's scalar cases. Literals give int/double/string/bool/null;
        // .uint / .float are reachable explicitly (values beyond Int range, 32-bit precision).
        case string(String), int(Int), uint(UInt64), double(Double), float(Float), bool(Bool), null
    }
}

public enum YAMLPath {
    public enum Component: Equatable, Sendable, ExpressibleByStringLiteral, ExpressibleByIntegerLiteral {
        case key(String)      // "server"
        case index(Int)       // 0
    }
}

public enum YAMLEditError: Error, Sendable {
    case malformedDocument(YAMLError)                      // parse failed — carries message + 1-based line/column
    case documentUsesAnchorsOrAliases                      // D6: can't locate an alias safely
    case pathNotFound(path: [YAMLPath.Component])
    case notASingleValue(path: [YAMLPath.Component])       // target is a map or sequence
    case unsupportedValueShape(path: [YAMLPath.Component]) // multi-line/block/folded scalar, null, or offset skew (a flow-collection *scalar* is editable)
}
```

**Pipeline** (all Swift, over the existing guarded bridge). Throughout, `let bytes = Array(yaml.utf8)` is the source (scanning + splicing index into it); parse and `valueSpan` run over the same UTF-8 via `yaml.withCString`, so `span.pos`/`span.end` are valid indices into `bytes`.
1. **Reject anchors/aliases (D6).** A quote/comment-aware source scan (modeled on `checkNesting`): at each node-content position (after `:`/`-`/flow-indicators/indent), **skip an optional tag** (`!…`) and whitespace, then flag `&` (anchor) or `*` (alias) → `documentUsesAnchorsOrAliases`. The tag-skip is required for the no-false-negative guarantee: YAML permits a tag before the property in either order (`!t &a`, `&a !t`) and the scanner tokenizes tags then continues (`scanner.cpp:147-148`), so a node-initial-only test would miss `!t &a`. Conservative — over-flags ambiguous spots (e.g. block-scalar bodies) rather than ever miss one; mid-scalar `&`/`*` (a URL's `a&b`, a glob's `*`) aren't at a node-content start, so they don't trigger.
2. **Parse, guarded.** `let r = yaml.withCString { yamlx.parse($0) }; guard r.ok else { throw YAMLEditError.malformedDocument(.parse(message: String(yamlx.parseMessage(r)), line: Int(r.line) + 1, column: Int(r.column) + 1)) }; let root = r.root`. (`yamlx.parse` returns a `ParseResult`, not a node; the wrapped `YAMLError.parse` keeps the 1-based position the shim reports.)
3. **Navigate** `path`, validating at every step — never index an undefined node (`mapValue`/`seqItem` return an undefined `Node()` on *any* miss — `yamlcpp_shims.h:172-213`): for `.key`, require `nodeKind(current) == map`, scan its pairs (`count`/`mapKeyText`) for an exact match and take that `mapValue`; a non-map parent or no match → `pathNotFound(path:)` (trail so far). For `.index`, require `nodeKind(current) == sequence` and `0 ≤ index < count`, then take `seqItem`; otherwise → `pathNotFound(path:)`. After taking a child, `nodeKind(child) == 0` (undefined) → `pathNotFound(path:)`.
4. **Classify** with `nodeKind`: a map or sequence → `notASingleValue(path:)`; a **null** (an empty value — `key:` with nothing after) → `unsupportedValueShape(path:)`, since it has no scalar span to overwrite and writing an absent value is an *insertion*, out of scope; only a scalar proceeds.
5. **Find the span.** `let span = yamlx.valueSpan(node, cstr, bytes.count)`; **guard `span.ok`** — a false `ok` means an invalid/unaddressable node (`span.pos == -1`), so throw `unsupportedValueShape(path:)` and never index with it. Then `let pos = Int(span.pos)`:
   - `span.endReliable` → plain single-line; `let end = Int(span.end)`.
   - else if `bytes[pos]` is `"` or `'` → **quoted**; scan for the close → `end` (below).
   - else → `unsupportedValueShape(path:)`.
6. **Serialize** the replacement: lower `YAMLEditor.Value` → `YAMLValue`, then `let token = YAMLSerialization.emitScalarToken(_:)` (internal, no trailing newline).
7. **Splice** the byte array and decode: `return String(decoding: bytes[0..<pos] + Array(token.utf8) + bytes[end...], as: UTF8.self)`.

**Quoted-end scan (D5), single-line only** — over `bytes` (`count = bytes.count`), from `pos` (the opening quote). Every read is bounds-checked (no OOB — CWE-125):
- **Double (`"`):** `var i = pos + 1; while i < count { let b = bytes[i]; if b == UInt8(ascii: "\\") { i += 2; continue }  // skip the escaped char, e.g. \" ; if b == UInt8(ascii: "\"") { return i + 1 }  // end (exclusive) ; if b == UInt8(ascii: "\n") { break }  // multi-line → defer ; i += 1 }`
- **Single (`'`):** `var i = pos + 1; while i < count { if bytes[i] == UInt8(ascii: "'") { if i + 1 < count && bytes[i + 1] == UInt8(ascii: "'") { i += 2; continue }  // escaped '' ; return i + 1  // end (exclusive) } ; if bytes[i] == UInt8(ascii: "\n") { break } ; i += 1 }`
- Falling out of either loop (EOF or a newline before the close) → `unsupportedValueShape` (unterminated / multi-line). The `while i < count` bound (and `i + 1 < count` before the single-quote lookahead) makes every index read in-bounds. Escape semantics per `exp.cpp:80-82` (`''`→`'`) and `:108-109` (`\"`→`"`).

**Serialization helper (A3)** — new **internal** `YAMLSerialization.emitScalarToken(_ value: YAMLValue) -> String`: the same per-kind logic the encoder's `write` uses for scalars — `.int`/`.uint` via `String(_:)`, `.double`/`.float` via `YAMLScalar.format`, `.string` plain or double-quoted per `needsQuoting` (via the emitter helpers) — **without** the trailing newline `emit` adds. `YAMLEditor` calls this so the spliced bytes equal what a full emit would produce.

## 4. Tests (`Tests/YAMLTests/`)

Hand-authored commented-config fixtures, asserted with the re-parse oracle:
1. **Minimal-diff preservation** — after `set`, every other line (comments, blanks, key order, untouched quoting) byte-identical; only the edited value's bytes changed.
2. **Oracle** — re-parse through yaml-cpp; the target reads back as the intended value.
3. **Plain target** (`port: 8080`→`9090`); **quoted target** (`tls: "off"`→`"on"`, stays quoted); **numeric-looking string** (`to: "2"`→quoted); **list-index path** (`["features", 0]`).
4. **Quoted-end scan** — embedded escapes: `k: "a\"b"` and `k: 'a''b'` edited, span covers the whole token; a multi-line quoted value defers.
5. **Errors** — `pathNotFound` (bad key, out-of-range index), `notASingleValue` (map/sequence target), `unsupportedValueShape` (multi-line/block/folded, null), `documentUsesAnchorsOrAliases` (a doc with `&a`/`*a`); each the right case with the path trail. Flow-collection *scalars* are covered as **editable** (plain, quoted, nested, multi-line-spanning, quote-as-needed), with refusal parity for null/multi-line-quoted inside flow.
6. **Round-trip sanity** — setting a value to its current value is a no-op diff.

## 5. Risks & mitigations

- **Anchors/aliases mis-splice (D6).** The node tree collapses alias→anchor, so an alias target's mark is the anchor's; a naive splice would silently edit the shared value. *Mitigation:* document-level refusal via the pre-parse scan (step 1), which **skips tags** so `!t &a` / `!t *a` aren't missed (`scanner.cpp:147-148`) and has no false negatives; tested. Ruamel-style per-alias preservation is future and needs a modeling DOM.
- **Quoted-end scan correctness.** Escapes are the trap. *Mitigation:* fixtures with embedded `\"` / `''`; newline/EOF before close → defer (single-line only this cut).
- **Splice/emit consistency.** Spliced bytes must equal an engine emit or the oracle disagrees. *Mitigation:* the shared internal `emitScalarToken` (A3), not a hand-rolled quoter; and no `YAMLScalar.format` on integers (it has only `Double`/`Float` overloads — integers use `String(_:)`).
- **Byte-offset skew (BOM).** `valueSpan` fails closed on a skew → `unsupportedValueShape`; tested.
